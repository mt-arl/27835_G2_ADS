# Líneas Base del Proyecto KairosMix

## Descripción del Proyecto
KairosMix es un sistema de gestión empresarial para la administración de productos y clientes, desarrollado con arquitectura MERN Stack (MongoDB, Express, React, Node.js) con TypeScript en el backend.

## Stack Tecnológico
- **Backend**: Node.js + Express + TypeScript + MongoDB + Mongoose
- **Frontend**: React 19 + Vite + JavaScript
- **Patrones de Diseño**: Singleton, Factory, Repository
- **Base de Datos**: MongoDB (NoSQL)
